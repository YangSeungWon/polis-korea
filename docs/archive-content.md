# 아카이브 페이지 콘텐츠 — 종류별

`/archive/{id}/` 페이지가 선거 종류(대선·총선·지선)별로 보여주는 정보. 템플릿·렌더러가
종류마다 다르다(`sync_archive_html.py` 생성, `assets/archive/{pres,general,local}.js` 렌더).
모든 데이터는 `data/results/{id}.json`을 런타임 fetch.

## 공통 — hero(상단)

모든 아카이브가 공유:
- **태그·선거명·날짜** + 정당별 의석/득표 요약 라인
- **scorecard** — 1·2위 비교 카드 (종류별 행 다름, 아래)
- **hero 메타** — 투표율, 박빙 N곳(격차 작은 순 5곳·클릭 시 이동), 빠른 링크(여론조사·출구조사·재보궐·종료)
- **상태 문구** — "개표 중" / "확정 결과 · 갱신시각"
- **/history 링크** — "역대 결과에서 hex로 →"

## 대선 (`pres.js`)

전국 1 race + 시도 17개 구도.

| 영역 | 내용 |
|---|---|
| hero scorecard | **후보 / 득표율 / 득표수** (1·2위) + 시도 우세 수 |
| **전국 결과** | 전국 후보 막대 + 시도별 1위 분포 |
| **출구조사 vs 실제** | 방송3사 출구조사 ↔ 개표 비교 |

## 총선 (`general.js`)

254 지역구 + 비례 47석.

| 영역 | 내용 |
|---|---|
| hero scorecard | **의석 반원(hemicycle, 300석)** + 지역구/비례/합계 (1·2당) |
| **의회 구성** | 선출직 정당 분포 — 지역구 winner + 비례 의석 정당별 |
| **비례대표 정당 득표** | 비례 정당별 득표·의석 |
| **출구조사 vs 실제** | 출구조사 ↔ 개표 |

## 지선 (`local.js`)

6 선출직(광역장·기초장·광역의원·기초의원, 의원은 지역구+비례).

| 영역 | 내용 |
|---|---|
| hero scorecard | **광역단체장·기초단체장·광역의원·기초의원** (지역구·비례·기타 정당 합산) |
| **선출직 정당 분포** | 6직위 정당별 막대(광역장·기초장·광역의원 지역구/비례·기초의원 지역구/비례) |
| **광역단체장 결과** | 17 시도 hex — 1위 후보·득표율, 정당색 |
| **시·도의회 의석 분포** | 시도 cluster hex — 광역의원 지역구+비례 의석 비례 배분. 클릭 → 당선인 필터 |
| **시군구의회 의석 분포** | 시군구 hex 안 의석 spiral — 기초의원 지역구+비례. 클릭 → 당선인 필터 |
| **시·도의원·시·군·구의원 당선인** | 전체 당선인 목록(검색·시도·정당·직급 필터·다단 그리드). 기본은 안내(지도 클릭/검색 유도) |
| **출구조사 vs 실제** | 출구조사 ↔ 개표(단체장) |
| **여론조사** | 그 회차 여론조사 페이지(`/governor` 등) 링크 |
| **재·보궐** | 동시 시행 재·보궐 결과(있으면) |
